# 危险标志识别系统：其他电脑运行说明

本文档说明如何把项目复制到另一台 Windows 电脑运行。当前版本只推荐使用 **Python 前端推理版**，启动时会自动加载训练好的 PyTorch 模型，不依赖浏览器 ONNX Runtime。

## 1. 必须保留的文件

运行演示至少需要这些内容：

```text
danger-sign-recognition/
  model/
    artifacts_viewpoint/
      best_danger_sign_model.pt
      training_config.json
      labels.json
    architectures.py
    constants.py
    evaluate_frontend_photos.py
    __init__.py
  python_frontend/
    assets/signs/
    server.py
    start_python_frontend.ps1
    README.md
  test_inputs/
  requirements.txt
  README.md
```

为了省事，也可以直接复制整个清理后的 `danger-sign-recognition/` 文件夹。

## 2. 安装环境

目标电脑建议使用：

```text
Windows 10/11
Python 3.10 - 3.12
Edge 或 Chrome 浏览器
```

如果还没有 Python，安装时勾选：

```text
Add python.exe to PATH
```

打开 PowerShell 检查：

```powershell
python --version
pip --version
```

## 3. 安装依赖

进入项目目录：

```powershell
cd C:\Users\你的用户名\Desktop\danger-sign-recognition
```

安装依赖：

```powershell
pip install -r requirements.txt
```

如果网络慢，可以使用清华源：

```powershell
pip install -r requirements.txt -i https://pypi.tuna.tsinghua.edu.cn/simple
```

没有独立显卡也可以运行，系统会自动使用 CPU。

## 4. 启动系统

```powershell
powershell -ExecutionPolicy Bypass -File python_frontend\start_python_frontend.ps1
```

正常启动时会看到类似信息：

```text
Loading trained model and serving Python frontend...
Checkpoint: ...\model\artifacts_viewpoint\best_danger_sign_model.pt
URL: http://127.0.0.1:7860/
loading model: ...
model loaded: arch=efficientnet_b0 device=cpu classes=5
open http://127.0.0.1:7860/
```

然后打开浏览器访问：

```text
http://127.0.0.1:7860/
```

PowerShell 窗口不要关闭。关闭后，网页服务也会停止。

## 5. 验证模型是否加载成功

打开：

```text
http://127.0.0.1:7860/api/status
```

如果看到类似结果，说明模型已经真正加载：

```json
{
  "loaded": true,
  "checkpoint": "model\\artifacts_viewpoint\\best_danger_sign_model.pt",
  "arch": "efficientnet_b0",
  "device": "cpu",
  "classes": [
    "explosion",
    "falling_objects",
    "falling_rocks",
    "flammable",
    "water_hazard"
  ]
}
```

## 6. 使用方式

网页打开后可以：

- 上传图片识别
- 拖拽图片识别
- 开启摄像头
- 单帧识别当前画面
- 实时连续识别

摄像头模式需要浏览器授权摄像头权限。

## 7. 常见问题

### PowerShell 提示无法运行脚本

使用下面的命令启动：

```powershell
powershell -ExecutionPolicy Bypass -File python_frontend\start_python_frontend.ps1
```

### 提示找不到 Python

重新安装 Python，并确认安装时勾选 `Add python.exe to PATH`。安装后关闭并重新打开 PowerShell。

### 提示没有 torch

```powershell
pip install -r requirements.txt
```

### 提示找不到模型文件

检查这个文件是否存在：

```text
model\artifacts_viewpoint\best_danger_sign_model.pt
```

如果不存在，说明复制项目时漏掉了模型文件。

### 端口 7860 被占用

可以换一个端口：

```powershell
python python_frontend\server.py --port 7861
```

然后访问：

```text
http://127.0.0.1:7861/
```

## 8. 重新训练

如果目标电脑还需要重新训练，先生成增强数据集：

```powershell
python model\build_viewpoint_dataset.py --source dataset --out dataset_viewpoint --per-class 220 --size 320
```

再训练：

```powershell
python model\train_model.py --dataset dataset_viewpoint --out model\artifacts_viewpoint --arch efficientnet_b0 --pretrained --freeze-backbone --epochs 30 --batch-size 16 --repeats 1
```

训练完成后重新启动：

```powershell
powershell -ExecutionPolicy Bypass -File python_frontend\start_python_frontend.ps1
```
